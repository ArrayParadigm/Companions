# Version 0.1.8

Follow this guideline on versioning.

*Every release this filename should be altered to reflect the new version.


MAJOR++

    Engine rewrite

    World reset

    Incompatible save data

    New product generation

SUBMAJOR++

    Expansion-level content

    Major system overhauls

    New classes, continents, or core mechanics

FEATURE++

    New features

    New quests, skills, items

    Gameplay-affecting changes

FIX++

    Bug fixes

    Typos

    Minor balance tweaks

    Non-breaking QoL
