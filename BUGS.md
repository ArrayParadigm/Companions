### Bugs
# Guidelines
- BUG format will be "## <version>-<bug number>
- If a bug is fixed it should be in a file called "BUG-FIX.MD", using the bug, with one # for "bug", one # for cause, and one # for solution.
    example:

    ## 0.1.3.3-003
    # BUG
        Every time someone types Potato in the command prompt the game shutsdown
    # Cause
        Early test added Potato for a shutdown command and carelessly forgot to remove it.
    # Solution
        Removed the command as it was not needed.
