RECRUIT REBUILD PT PLAN
Core + Endurance Focus
Prepared by: Eve / Tiny Tyrant Clipboard Division

MISSION
Rebuild the chassis. Strengthen the core. Improve endurance. Restore consistency.
No random hero workouts. No vague “I should exercise later.” This is scheduled training.

PRIMARY FOCUS
1. Core stability
2. Endurance
3. Functional strength
4. Mobility and recovery
5. Consistency over ego

SAFETY RULE
Follow any VA, doctor, or physical therapist restrictions.
Stop the session if pain becomes sharp, unsafe, dizzying, chest-related, nerve-like, or medically wrong.
Discomfort is allowed. Stupidity is not.

WEEKLY SCHEDULE

Monday:
Main PT Alpha — Core + Endurance

Tuesday:
Daily Minimum — Beloved Creature Maintenance

Wednesday:
Main PT Bravo — Strength Chassis + Core

Thursday:
Recovery Mobility + Check-In

Friday:
Main PT Charlie — Endurance + Movement Discipline

Saturday:
Optional light walk, technique, stretch, or mobility

Sunday:
Rest, review, plan next week

Core + Endurance

Purpose: Primary rebuild day. Core first, engine second.

Endurance:
